# 🗒️ GoldenTicket
프로젝트 기간 : 2023.09.24 ~ 2023.11.30

# 🧗 프로젝트 소개
- 각종 영화 정보를 확인하고 리뷰를 달 수 있습니다.<br>
- 영화에 매겨진 평점을 통해 영화의 순위를 확인할 수 있습니다.<br>
- 마음에 드는 영화를 북마크하여 저장해 둘 수 있습니다.<br>
- 커뮤니티를 통해 소통이 가능합니다.
